  • Siapkan langkah-langkah untuk mengekstrak BIOS dari PS2 Anda, seperti menggunakan homebrew atau perangkat lunak khusus (misalnya uLaunchELF).

  • Simpan file BIOS (biasanya bernama seperti "scph10000.bin") di lokasi yang mudah diakses.

  • Setelah mendapatkan BIOS, Anda harus mengatur jalur file-nya di emulator (biasanya di menu konfigurasi BIOS).
